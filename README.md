## Description

This is a api for province of VietNam. This api is built with NestJS and TypeORM.
## Installation

```bash
$ npm install | yarn install | pnpm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
